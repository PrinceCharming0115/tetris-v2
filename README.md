# Puyo Puyo Tetris II

# Current Features
- Colors reflect actual Tetris
- Able to clear lines
- Able to game over
- Keeps score
- Able to hold onto pieces

# Upcoming Features
- Piece forecast
- Ghost pieces
- Quick Drop

Bootstrapped using this tutorial: https://youtu.be/H2aW5V46khA
Edits, improvements, and additional features added by me.
